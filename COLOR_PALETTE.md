# COLOR_PALETTE.md

## Core Brand Palette (2025 Refresh)

Based on the "Paper & Circuitry" Design System.

| Color Name | Hex Code | Design System Variable | Usage |
| :--- | :--- | :--- | :--- |
| **Navy (Primary)** | `#15354A` | `--color-primary` | Primary buttons, active states, key text accents. |
| **Slate (Secondary)** | `#495A68` | `--color-secondary` | Headers, navigation, card borders, icons. |
| **Soft Gray** | `#D4D8DC` | `--color-soft-gray` | Backgrounds, dividers, neutral chips. |
| **Cream** | `#F5F1E8` | `--color-cream` | Light surfaces, paper feel, hero overlay (light). |
| **Soft Green** | `#7AA881` | `--color-accent` | Success states, AI highlights, glowing nodes. |
